# 行业热度增强版选股系统

## 功能概述

本系统在原有 B1 完美图形匹配选股的基础上，增加了**行业热度分析**功能，帮助你：
- 避免选到冷门行业的股票
- 优先选择热门赛道的优质股
- 提高选股的成功率

## 核心原理

回测分析发现：
- **高相似度 + 高热度** 组合的股票胜率最高
- 冷门行业的股票即使图形匹配度高，表现也往往不佳
- 行业热度 = 0.6 × 成交额占比 + 0.4 × 成交额环比

## 使用方法

### 方式 1：独立脚本（推荐，不影响原有功能）

```bash
# 使用默认参数（相似度≥60%，回看 25 天）
python select_with_heat.py

# 提高相似度阈值到 70%
python select_with_heat.py --min-similarity 70

# 调整回看天数为 30 天
python select_with_heat.py --lookback-days 30

# 快速测试（只处理前 500 只股票）
python select_with_heat.py --max-stocks 500

# 调整碗口策略回溯天数
python select_with_heat.py --M-days 35
```

### 方式 2：原有命令（不带行业热度）

```bash
# 原有命令仍然可用，但不显示行业热度
python main.py run
```

## 决策建议

### 选股优先级

1. **首选**：高相似度 (≥92%) + 高热度 (≥70 分)
   - 胜率最高，回测数据显示可达 65%+

2. **次选**：高相似度 (≥92%) + 中等热度 (50-70 分)
   - 图形匹配好，行业不拖后腿

3. **谨慎**：低相似度 (<75%) 或 低热度 (<40 分)
   - 回测数据显示胜率显著较低

4. **避免**：双低（低相似度 + 冷门行业）
   - 胜率最低，建议直接跳过

### 持有期建议

根据回测分析（查看 `backtest_results/` 目录中的 MD 报告）：
- **最佳止盈窗口**：通常在选入后 7-10 天
- **止损建议**：跌破 -4% 时考虑止损（回测显示跌破 -4% 后只有约 20% 能盈利）

## 常见问题

### Q: 行业热度是如何计算的？
A: 行业热度 = 0.6 × 成交额占比分位数 + 0.4 × 成交额环比
   - 成交额占比：行业成交额 / 全市场成交额
   - 成交额环比：(今日成交额 - 昨日成交额) / 昨日成交额

### Q: 为什么有些股票显示"未知"行业？
A: 可能原因：
   1. akshare 未安装，使用内置映射（只覆盖约 50 只主要股票）
   2. 该股票不在行业成分股列表中

   建议安装 akshare：`pip install akshare`

### Q: 相似度阈值设置多少合适？
A: 回测建议：
   - 60-70%：选股数量多，但胜率较低
   - 70-85%：平衡选择
   - 85-95%：选股数量少，但胜率高

### Q: 多久运行一次选股？
A: 建议每个交易日收盘后运行（15:30 之后）
   - 数据已更新
   - 可以及时获取最新行业热度
